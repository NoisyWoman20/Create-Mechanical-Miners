# 机械动力：机械矿机 / Create: Mechanical Miners

## 此仓库为 *NeoForge* 版本

---

## 简介 / Introduction

本模组让你能利用矿机，在 Minecraft 中无限地产出矿物。
> This mod lets you harness your mining machine to produce ores endlessly in Minecraft.
